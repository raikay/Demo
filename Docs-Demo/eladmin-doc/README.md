# EL-ADMIN-DOC

EL-ADMIN 文档源码

## Build Setup
``` bash
#下载node.js，使用npm安装
npm install -g yarn

#查看版本：
yarn --version

# 安装依赖
yarn install

# 启动服务
yarn docs:dev

# 构建生产
yarn docs:build
```
